# Brain-aligned-visual-encoding-via-feature-pruning
This project aligns computer vision models (DINOv2, CLIP) with human fMRI responses. Building on VISGate and BOLDREAMS, it tests if ROI- and semantics-aware feature pruning creates compact, interpretable brain encoders. It evaluates the trade-off between model size and predictive accuracy, with potential subject-aware and gating extensions.
